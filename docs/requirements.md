## 業務手順

- 占い師のユーザー登録を行う
  - Microsoft Entra External IDを利用する
- フォーム作成画面へ進む
  - フォームはデフォルトで氏、名、メールアドレス、生まれた年・月・日、生まれた時・分、生まれた場所の国・都道府県・市町村を含む
  - メールアドレスはバリデーションを含む
  - 全ての項目は削除可能
  - 全ての項目の項目名は変更可能
  - カスタム項目としてメールアドレス、年月日、時分、国、都道府県、市町村、テキスト、数値、自由入力欄を含む
